Koi Fish Vans

These vans are a personal project that I created during quarantine. I drew the design and used a Shilouette machine to cut the pattern on vynil heat transfer. They are currently for sale on Etsy.

Poppy Poster

I designed this poster for the shared space in my house using Photoshop. I hand drew the flowers using the brush tool and implemented several different layers to get the wacy effect that I wanted to incorporate.

Space Poster

I created this poster based off of a picture I took and wanted to display in my room. I experimented with different tools in Photoshop such as the blur and liquify tools to test out different effects, which ended up being part of the final design.

Durango Poster

This poster was a gift for a friend who grew up in Durango. I used several layers and text to create this poster and experimented with different brushes for the sky.

OutKast Poster

This poster was the most challenging for me to create and really tested my Photoshop skills. I had to use several filters and hand draw several elements to get the final image I wanted. I also spent a lot of time on the color scheme.
