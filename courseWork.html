  • Final JavaScript Project: Word Collage Generator

    This is my final project for Web, where I created a tool that lets you input a paragraph and generate a colorful collage where the size of the words is based on the frequency of their occurence. I wrote a JavaScript program that takes the paragraph input, parses the string into words that are stored in an array. I then iterated through this array and converted each word to lower case for comparison, storing each unique word in a new array of objects containing a word and the number of times the word was found. Then, I created another function that iterated through this new array and incremented the font size for each occurence of a word, making the more frequently appearing words appear larger. I also implemented a way for the color and placement of the words to be random, to make it more interesting.
