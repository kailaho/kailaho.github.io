This project is a classroom accessory designed to help children with austim relax and calm their senses by sorting shapes and creating music. The combination of light and sound sensory features allow them to decompress while also learning as the tool guides them through learning a piano song by following lights and pressing on the fish.

Sensory Aquarium

Process Details

In creating this project, I wanted to create something that had the capability to have an impact for children who may struggle to learn in a traditional setting. My freshman year, I worked on a group project with a teacher for students with autism, and she requested that we design a device that helps her students stay focused and decompressed in the classroom. Our solution was a hand-compression device, as compression of certain body parts has been proven to help children with autism remain calm. I was inspired by this project to try to create another device, this time one that utilizied my skills in physical computing, that would help these students decompress in a classroom setting. I came up with the sensory aquarium, which uses a combination of Ardiuno code, LED lights, and a piezo buzzer to create a visual and auditory experience for the users. First, students sort the fish into their cut out positions, as sorting is an activity proven to help those with autism decompress. Once the fish are placed, the user can press on the fish which each emit a different piano note. There are two modes to toggle between: one that allows the user to play the notes freely, and one that guides the user through learning a piano song by lighting up the LED lights above the fish coordinating to the note.

As part of the form final project, I designed a chess set that is both elegant and portable. It is meant to be simple and easy to manufacture, but still have an appealing and functional design.

Chess Set

Process Details

For this project, the main focus was demonstrating my 3D modelling skills in Sketchup in order to design a chess set that met the requirements of being easy to transport, and have a box for storing the pieces. I wanted to make an elegant and easy to use chess set, and since it had to store pieces, I wanted to take it one step further and make it travel-friendly by creating a design where the pieces snap onto the board in order to prevent them from falling. I used a laser cutter and thin plywood sheets to create every element of the board, and designed each piece in Sketchup. The pieces are slotted together since I was limited by the 2D dimension of the plywood, and everything is wood-glued together. There is a drawer with a leather strap that allows the user to store the pieces. This project was a good test of my 3D modelling and design skills, as I had to be creative and resourceful in order to create a deisgn that fit both the constraints and I was happy with given the materials and tools I had available. I learned a lot about the fabrication process, design process, and prototyping prcoess in creating this piece, as I had never worked with 3D modeling software before. I found that while it was a bit tedious at times, it is an extremely powerful tool.

This was a group project (Seren Chapin, Campbell Mahan) in which we were tasked in creating an app based on observations we made on how we spent our alone time. We created a mockup for a mindfulness app that helps the user relax and engage in mindful activities in an engaging and fun way.

Alone Time - Group Project

Working in groups, we were challenged to create a product that provides a solution for an issue that we observed about how we spend our alone time. We observed individually and then discussed that often times when we are alone, we feel the constant need to fill our time with some sort of media or entertaiment. We wanted to create a solution to this that provided a way for users to consume media while still being productive and unwinding. Working through the design process by brainstorming, creating fictional personas, and prototyping, we came up with "Attentio," a mindfulness app centered around growing virtual plants as the user completes certain mindfulness challenges. Our final product was an Adobe XD mockup that demonstrates the functionality of the app. This project was very enlightening in how different the design process is when working in a group than when working alone. It allows for a broader range of perspective and ideas, which brings about a solution that is often more fitted to the needs of the users and their goals. This project was an important learning milestone for working in design teams, and taught me a lot about how to communicate and express creative ideas effectively.

Using physical computing and the Arduino IDE, I created a color changing lamp with different modes. The lamp is meant to be aestheitcally pleasing and playful.

Color-changing Lamp

This project is meant to demonstrate the use of analog inputs that translate to a digital output. In this case, the user can turn two knobs on the outside of the lamp: one to change the modes (between continuous color cycling or choosing a static color) and one to choose the color should they choose to have a static color. I liked the idea of colored lights as the output, but found that the light from the LED strip was too harsh. I was inspired by paper lanterns that are often used as dectoration, and thought that paper would be a great way to diffuse the light. Additionally, I didn't want a simple rectanglar or cubical lamp, so I designed a cash-register-like shape and hand made the frame for the lamp in the wood shop to be the exact size of the breadboard that housed the Arduino. This project was a test of my aestheic design skills, but more so my physical computing skills as I struggled with some hardware aspects of the piece. I found that due to my extensive experience in C++, coding in the Arduino IDE was a pleasant experience and challenged me to think about coding in a new way that extended beyond the typical use of C++ coding.
