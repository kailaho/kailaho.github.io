Sensory Aquarium

Sensory Aquarium is a classroom accessory designed to help children with austim relax and calm their senses by sorting shapes and creating music. The combination of light and sound sensory features allow them to decompress while also learning as the tool guides them through learning a piano song by following lights and pressing on the fish.

Demonstrated Skills

  • Arduino IDE
  • Product Development
  • Physical Computing
  • C++ Programming

Precision Play Chess Set

Precision play chess set is designed to be both elegant and portable. Created using 3D modeling software and laser cutting, it is meant to be simple and easy to manufacture but still have an appealing and functional design.

Alone Time - Group Project

Alone Time is a group project (Seren Chapin, Campbell Mahan) in which we were tasked in creating an app based on observations we made on how we spent our alone time. We created a mockup for a mindfulness app that helps the user relax and engage in mindful activities in an engaging and fun way by completing mindfulness tasks to water plants and grow their virtual garden. My specific role was in designing the building and visuals for the main page of the app.

Demonstrated Skills

  • Adobe Photoshop
  • Wireframing
  • Storyboarding
  • Collaboration
  • UX/UI Design

Color-changing Lamp

Created using physical computing and the Arduino IDE, the color-changing lamp has different modes the user can choose from, either cycling through the colors repeatedly, or choosing a static color using a dial on the side of the lamp. It is meant to be aestheitcally pleasing and playful.
